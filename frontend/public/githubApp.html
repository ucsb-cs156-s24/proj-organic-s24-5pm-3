Installing github app

  1. Open the following link in a new tab: https://github.com/apps/ucsb-cs-linker
  2. Click on the "Install" button (if you have previously installed the app, you will see "Configure" instead of "Install")
  3. Select the organization you want to install the app for
  4. Confirm application permission by clicking "Install"
